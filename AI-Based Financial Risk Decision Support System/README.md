# AI-Based Financial Risk Decision Support System

This project is a machine learning powered web application that predicts financial risk using user financial data.

The system analyzes income, debt, and credit score to classify financial risk and simulate different financial scenarios.

## Features

- Financial risk prediction using machine learning
- Scenario simulation for financial decisions
- Financial analytics dashboard
- Data visualization with charts
- Modular project architecture

## Technologies Used

- Python
- Flask
- Scikit-learn
- Pandas
- HTML
- CSS
- JavaScript
- Chart.js
